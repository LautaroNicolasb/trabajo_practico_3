Introducción a HTML

HTML (HyperText Markup Language) es el lenguaje estándar utilizado para crear y estructurar contenido en la web. Es un lenguaje de marcado que define la estructura de las páginas web utilizando una serie de elementos (etiquetas) que delimitan y organizan diferentes tipos de contenido, como texto, imágenes, enlaces, formularios y otros componentes interactivos.

Características de HTML

  1. Lenguaje de Marcado: HTML es un lenguaje de marcado, lo que significa que utiliza etiquetas para "marcar" elcontenido de un documento. Cada etiqueta tiene un propósito específico y define diferentes tipos de contenido.
  2. Etiquetas: HTML está compuesto por etiquetas (tags) que son instrucciones que el navegador utiliza para mostrar el contenido. Las etiquetas HTML son palabras clave encerradas entre corchetes angulares (< >), y generalmente vienen en pares de apertura y cierre, como <p> y </p> para los párrafos.
  3. Atributos: Las etiquetas HTML pueden tener atributos que proporcionan información adicional sobre los elementos. Los atributos se especifican en la etiqueta de apertura y tienen el formato nombre="valor". Por ejemplo, a href="https://www.example.com"Enlace /a.
  4. Estructura Jerárquica: HTML tiene una estructura jerárquica y anidada. Los elementos HTML pueden contener otros elementos, creando una estructura en árbol.
  5. Semántica: HTML5 introdujo un conjunto de etiquetas semánticas que mejoran la estructura y el significado del contenido, como header, footer, article, section, y nav. Estas etiquetas hacen que el contenido sea más accesible y comprensible tanto para los motores de búsqueda como para los lectores de pantalla.
  6. Independencia de Plataforma: Los documentos HTML pueden ser visualizados en cualquier dispositivo y sistema operativo con un navegador web, lo que lo hace independiente de la plataforma.
  7. Interactividad: Aunque HTML por sí solo no es interactivo, se puede combinar con CSS (Cascading Style Sheets) y JavaScript para crear páginas web interactivas y atractivas. CSS se utiliza para el diseño y la presentación visual, mientras que JavaScript añade comportamiento dinámico e interactividad.
  8. Extensibilidad: HTML se puede extender mediante el uso de lenguajes como XML y mediante la integración con tecnologías web modernas como SVG para gráficos vectoriales y MathML para contenido matemático.
  9. Compatibilidad hacia Atrás: HTML es compatible con versiones anteriores, lo que significa que los navegadores modernos pueden renderizar contenido HTML escrito en versiones más antiguas del lenguaje.

Elementos

Algunos elemento básicos de HTML incluyen:

  1. <!DOCTYPE>: Define el tipo de documento y la versión de HTML que estás utilizando.
  2. <html>: Define el inicio y el final del documento HTML.
  3. <head>: Contiene información sobre el documento, como el título, enlaces a estilos CSS, metadatos, etc.
  4. <title>: Define el título del documento que se muestra en la pestaña del navegador.
  5. <body>: Contiene todo el contenido visible del documento.
  6. <h1>, <h2>, ..., <h6>: Define encabezados de diferentes niveles, siendo h1 el más importante y h6 el menos.
  7. <p>: Define un párrafo.
  8. <a> Define un enlace.
  9. <img> Inserta una imagen.
  10. <ul>, <ol>, <li>: Define listas sin orden, ordenadas y elementos de lista, respectivamente.
  11. <div>: Define una sección genérica o contenedor en el documento.
  12. <span>: Define una sección en línea en el documento.
  13. <table>: Define una tabla.
  14. <tr>: Define una fila dentro de una tabla.
  15. <td>: Define una celda de datos dentro de una fila de tabla.
  16. <th>: Define una celda de encabezado de columna o fila dentro de una tabla.
  17. <form>: Define un formulario para recopilar datos del usuario.
  18. <input>: Define un campo de entrada en un formulario.
  19. <textarea>: Define un área de texto multilínea en un formulario.
  20. <button>: Define un botón clickeable.

Estructura

Elemento Descripción
<!DOCTYPE html> Declaración que define el tipo de documento como HTML5.
<html> Envuelve todo el contenido de la página.
<head> Contiene información sobre la página.
<body> Contiene el contenido visible de la página.
